Programação

Não é só de Front-End que vive a Tecnologia da Informação, esta área também abrange o que os olhos do usuário não vêem. O Back-End é tudo aquilo que o usuário não tem contato direto, como um banco de dados, por exemplo. É lá que a magia da programação acontece de fato. Existem diversas linguagens de programação, nós do Manual do Estudante recomendamos a linguagem Python para iniciantes, clique aqui para se aventurar.
